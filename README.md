# Nuxt Prismic Boilerplate

# Setup

``` bash
# install dependencies
$ yarn

# serve with hot reload at localhost:3000
$ yarn dev

# static build
$ yarn generate
```

# Optional Commands 

``` bash
# install plop globally
$ npm install -g plop

# bootstrap a new component or page
$ plop 
```
